# Folder Logos

Letakkan file logo PNG di sini dengan nama sebagai berikut:

## Daftar File yang Dibutuhkan:

- `canon.png` - Logo Canon
- `fujifilm.png` - Logo Fujifilm
- `samsung.png` - Logo Samsung
- `gopro.png` - Logo GoPro
- `olympus.png` - Logo Olympus
- `fujifilm2.png` - Logo Fujifilm alternatif
- `iphone.png` - Logo iPhone/Apple

## Persyaratan:

✅ Format: PNG dengan background transparan (RGBA)
✅ Nama file: Semua huruf kecil (lowercase)
✅ Ekstensi: `.png` (bukan `.PNG` atau `.jpg`)
✅ Ukuran: Bebas (akan auto-resize)

## Cara Download Logo:

1. Cari di Google: "[brand name] logo png transparent"
2. Atau gunakan situs seperti:
   - https://www.pngwing.com/
   - https://pngtree.com/
   - https://freepnglogos.com/

## Catatan:

Jika file logo tidak ada, aplikasi akan menampilkan nama brand sebagai text (fallback).
